# PatternFinder

**Track recurring events. Spot patterns.**

## The Problem

Many things in life happen in patterns — a pet's health issues, mood changes, habits, symptoms. But without tracking, it's impossible to see the pattern. You forget when it last happened, how often it occurs, and whether it's getting better or worse.

## The Solution

PatternFinder is a mobile-first PWA for tracking recurring events and spotting patterns over time. Create custom events (like "Dog diarrhea" or "Headache"), then log occurrences with a satisfying double-tap. The dashboard shows frequency charts across 24 hours, 7 days, or 30 days, helping you spot trends you'd otherwise miss.

## Who It's For

- Pet owners tracking health events (vet visits become much more productive with data)
- People monitoring personal health symptoms or habits
- Couples or families who want to track things together with shared events

## Key Features

- Google sign-in
- Create custom events with emoji and color coding
- Double-tap to log an occurrence with confetti animation
- Dashboard with frequency charts (daily, weekly, monthly views)
- Share events with specific people by email for collaborative tracking
- Real-time sync across devices
- Installable as a PWA on mobile

## Tech Stack

Next.js, Supabase (Postgres + Auth + Real-time), Vercel, Cloudflare DNS, PWA
